Staging

git add <filename>
	-stage commits
    - (to add all files) git add .

Commit

git commit -m “message”           
	-commit changes

Push setup

git remote add origin <link>         
	-creates remote connection
git remote -v          
	-shows remote links

Push

git push origin master              
	-push to remote repo 
	- shortcut to set upstream (git push -u origin master)
